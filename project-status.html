Go-live checklist (Shopify Admin settings, no code)
  • Shopify Payments activated — bank account connected and verified in Shopify Admin
  • Shipping zones & rates — configured for all regions you're selling to
  • Tax settings — correct for your region (Bangladesh / international)
  • Production environment variables — store domain and access tokens set on the server
  • Webhook registered — order sync to CRM pointed at the live domain
  • Custom domain live — pointed to deployment and SSL active
  • Test purchase — place a real test order end-to-end before announcing launch
